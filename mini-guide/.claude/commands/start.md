# /start - Begin Session

Lightweight startup. Load context and give a brief status update.

## Instructions

### 1. Load State (parallel)
- Run `date +%Y-%m-%d` for today's date
- Read `state/current.md`
- Read today's session log `sessions/{TODAY}.md` if it exists, otherwise the most recent file in `sessions/`

### 2. Brief Status (3 lines max)
- Date and day of week
- Top 2-3 priorities from current.md
- Where we left off (from session log)

No lengthy briefing. No email scan. Just orient and wait for instructions.
