# /update - Quick Checkpoint

Save progress without ending the session.

## Instructions

### 1. Append to Session Log
```markdown
## Update: {TIME}
- {what was worked on, 1-3 bullets}
```

### 2. Update State (if needed)
Only update `state/current.md` if something material changed.

### 3. Confirm
One line: "Checkpointed: {brief description}"
